# 🦤 DODO - AI 기반 일정 관리 플랫폼

> 단순 일정 기록을 넘어,  
AI가 하루를 요약하고 습관 형성을 돕는 **지능형 라이프 관리 서비스**

---

## 💡 프로젝트 개요

기존 일정 관리 서비스는  
사용자가 직접 계획하고 관리해야 하는 **수동적인 구조**라는 한계가 있습니다.

DODO는 이러한 문제를 해결하기 위해  
AI 기반 일정 브리핑과 사용자 맞춤형 기능을 결합하여  

👉 **“사용자의 하루를 능동적으로 가이드하는 서비스”**를 목표로 개발되었습니다.

---

## 🎯 핵심 목표

- AI를 활용한 **능동적 일정 관리 경험 제공**
- 생산성과 감성을 결합한 **커스터마이징 UI**
- 루틴/습관 데이터를 통한 **지속 가능한 자기관리 환경 구축**

---

## ✨ 주요 기능

### 📊 홈 대시보드
- 오늘 일정 AI 브리핑 제공
- 위젯 기반 커스터마이징 UI

### 🔁 습관 & 루틴

- 알림 기능 지원

### 🛒 스토어 & 커스터마이징
- 테마, 스티커, 글꼴 구매 및 적용

### 👤 사용자 관리
- 회원가입 / 로그인
- 소셜 로그인 (Kakao, Google)

## ⚠️ Trouble Shooting

### 1. 일정 데이터 통합 관리 문제

- 문제  
  일정, 할 일, 습관 등 다양한 데이터를 하나의 캘린더에서 통합 관리해야 하는 요구사항 발생

- 원인  
  데이터 구조가 분리되어 있어 조회 시 복잡도 증가

- 해결  
  공통 필드를 기준으로 데이터 구조를 재설계하고,  
  통합 조회 API를 구성하여 캘린더에서 일관된 데이터 제공

- 배운 점  
  초기 DB 설계가 서비스 확장성과 유지보수성에 큰 영향을 미친다는 것을 경험
